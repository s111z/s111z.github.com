无序数组求中位数（O(1)）

· 2023-10-08 ·

此类题目一般是给定一个无序的数组，在录入的过程中，不断询问中位数（这是题目本质，当然不同的题目表述的意思不同，不过大体都是这个意思。）
有四种算法
1.定义一个vector<>,每次插入数字前先用upper_bound()或者lower_bound()查找，再用insert（）插入。

vector<int>a;
while()
{
    int num;
    cin>>num;
    auto it=lower_bound(a.begin(),a.end(),num);//lower_bound()不仅可以用来查vector,也可以用在数组中
    a.insert(it,num);
}

2.multiset
维护两个multiset的结构s1,s2，两个都是从小到大排的，所以我们只需要将保证s1.size()>s2.size(),s1.size()-s2.size()<=1,同时 \s2.begin()>= *s1.rbegin();那么s1.rbegin()一定是中位数

multiset<int>s1,s2;
while()//录入数组的过程
{
    cin>>num;
    if(!s1.size()||num<*s1.rbegin()) s1.insert(num);
    else s2.insert(num);
}
while(s1.size()>s2.size()+1)
{
    s2.insert(*s1.rbegin());
    s1.erase(--s1.end())//s1.rbegin()不能当作迭代器用在erase中
}
while(s1.size()<s2.size())
{
    s1.insert(*s2.begin());
    s2.erase(s2.begin());
}
